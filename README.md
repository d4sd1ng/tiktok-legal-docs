# TikTok Legal Documents

This repository contains the Terms of Service and Privacy Policy for the YouTube Automations TikTok API integration.

## Files

- `terms-of-service.html` - Terms of Service document
- `privacy-policy.html` - Privacy Policy document
- `robots.txt` - Robots.txt for web crawlers

## URLs

Once GitHub Pages is enabled, these documents will be available at:

- Terms of Service: `https://d4sd1ng.github.io/tiktok-legal-docs/terms-of-service.html`
- Privacy Policy: `https://d4sd1ng.github.io/tiktok-legal-docs/privacy-policy.html`

## Setup

1. Enable GitHub Pages in repository settings
2. Select branch: `main`, folder: `/ (root)`
3. Wait 1-2 minutes for deployment
4. Use the URLs above in TikTok Developer Dashboard
